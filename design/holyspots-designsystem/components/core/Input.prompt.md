HolySpots text field — admin tables/forms, filter bars, and the login card.

```jsx
<Input label="Name" serif placeholder="Spot name" />
<Input variant="filter" placeholder="Search…" />
<Input variant="login" label="Login" />
<Input error="The Name field is required." />
```

`serif` reproduces the admin's serif data-entry face. Errors render in #ff5e5e at 14px.
