# HolySpots mobile UI kit

Interactive recreation of the 2012 iOS/Android app (sources: Android `res/layout/*.xml`, iOS ViewControllers). Flow: Spots/Guides home → region → spot detail (rate, check in) → reviews. Content is real data from `holyspots-db-v2012` (regions, spot names, review texts). 390px phone canvas; the same components reflow to the tablet split view described in `guidelines/responsive.card.html`.
